# 💸 Monitoring Uang Kas - XI RPL 3
> **Sistem Manajemen Keuangan Digital Kelas**

---

### 🎨 Visual Preview (Custom UI)
Proyek ini menggunakan **Tailwind CSS** dengan palet warna *Vivid Azure* dan *Slate* untuk memberikan kesan modern, bersih, dan profesional.

| Section | Style Highlights |
| :--- | :--- |
| **Header** | Rounded-3xl, Gradient/Solid Vivid Azure (#0d65a0), Soft Shadow |
| **Cards** | Border Slate-100, Hover Transition Effects |
| **Typography** | Plus Jakarta Sans, Heavy Bold for Emphasis |
| **Badges** | Capsule-style Status (Lunas/Belum) dengan High Contrast |

---

### 🚀 Fitur Utama
* ✅ **Real-time Data Fetching**: Menampilkan seluruh siswa (meskipun belum bayar).
* ✅ **Search Functionality**: Pencarian nama siswa secara instan.
* ✅ **Smart Status System**: Indikator otomatis untuk status pembayaran.
* ✅ **Responsive Design**: Tampilan optimal di berbagai ukuran layar.
